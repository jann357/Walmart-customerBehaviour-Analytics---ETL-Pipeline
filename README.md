The ETL Workflow Automation project was designed to implement a scalable,
automated, and enterprise-grade data engineering pipeline using Microsoft Azure cloud
services. The project focused on building a multi-source ETL workflow using Azure
Databricks and Azure Data Factory (ADF) integrated with Medallion Architecture
principles.
The solution automates the complete lifecycle of data ingestion, transformation,
orchestration, monitoring, and analytics generation. The workflow was developed using
parameterized Databricks notebooks and orchestrated through Azure Data Factory
pipelines with built-in scheduling, retry mechanisms, dependency handling, monitoring,
and alerting functionalities.
